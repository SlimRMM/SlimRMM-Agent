Installation Steps

1. The installer will copy the SlimRMM Agent to your Applications folder.
2. After installation completes, you will be asked for:
  • Server URL - Your SlimRMM server address (e.g., https://rmm.example.com)
  • Enrollment Token - The token from your SlimRMM dashboard
3. The agent will register with your server and start automatically.

Permissions

For remote desktop functionality, you may need to grant permissions in System Settings:

  • Privacy & Security → Screen Recording → SlimRMM
  • Privacy & Security → Accessibility → SlimRMM
